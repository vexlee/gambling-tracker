# Gambling Tracker

A real-time gambling session tracker built with React, Vite, and Supabase.

## Features
- Create and join game sessions.
- Real-time updates for players and bankers.
- Track buy-ins, winnings, and losses.
- Banker dashboard to manage the game.

## Tech Stack
- Frontend: React + Vite
- Styling: Tailwind CSS
- Database/Real-time: Supabase

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your Supabase environment variables in `.env`.
4. Run the development server: `npm run dev`
